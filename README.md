# Pokédex
Desarrollado por:
-  Meza Cruz Gael Alfredo

Aplicacion web interactiva sobre Pokemones, estadisticas base, tipos,etc utilizando la API oficial de PokeAPI

  
## Tecnologías utilizadas
- Se utilizaron tecnologias tales como React, Vite, Taildwin y Typescript.
  
## Instrucciones de instalación.
Clonar el repositorio:
```bash
git clone <URL_DEL_REPOSITORIO>
```
Ingresar al directorio del proyecto:
```bash
cd nombre-del-proyecto
```
Instalar dependencias:
```bash
pnpm install
```
Comando para ejecutar el proyecto
```bash
pnpm run dev
```  
## Funcionalidades implementadas.

- Lista de pokemones
- Detalles por pokemon
- Busqueda de pokemones
* Filtrado por tipos de pokemones
* Comparacion entre pokemones
* Sistema de favoritos
